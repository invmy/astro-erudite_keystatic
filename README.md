# astro-erudite_keystatic

Revised version with keystatic content collection added. Keystatic focuses on visibility-based accessibility. No component adaptation planned. Only Markdown text supported; HTML code and components are not supported.

[astro-erudite](https://github.com/jktrn/astro-erudite)

sub-posts slugs do not support auto-fill and require manual entry.
